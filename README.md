# 🎨 Color Clicker

A timed puzzle game where you fill a board with colors by clicking empty cells. Each click triggers a chain reaction that fills multiple fields with the same random color. Race against the clock to fill the entire board!

## 🎮 How to Play

1. **Click** any empty (gray) field on the board
2. A random color is generated and applied to:
   - The field you clicked
   - **3 additional random empty fields** (chain reaction!)
3. Each field gets a number showing the **order** it was filled
4. **Goal:** Fill every field on the board as fast as possible!

## ✨ Features

- **Dynamic board sizes** - Play on 5x5, 10x10, 15x15, or 20x20 grids
- **Real-time timer** - Tracks your completion time in milliseconds
- **Visual feedback** - Each filled field shows its click number
- **New Game menu** - Start fresh with any board size without restarting the app
- **Random colors** - Every click generates a unique RGB color
- **Win notification** - Popup dialog congratulates you with your final time
- **🎯 Color Preview** - Hover over any empty field to see the next color before clicking
- **🏆 High Score Tracking** - Best times saved locally for each board size
- **🔊 Sound Effects** - Satisfying click tones that pitch up as you progress
- **♿ Color Blindness Mode** - Toggle shape symbols (● ■ ▲ ★ ◆) for accessibility
- **✨ Chain Reaction Animation** - Visual feedback when fields fill automatically
- **📁 Persistent Storage** - High scores saved automatically to your home directory
